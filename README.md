# ☀️ 역광 + 한국 계절별 태양 시뮬레이터

야외 인물 촬영 시 태양 위치·고도에 따른 빛의 변화를 시각적으로 시뮬레이션하는 도구입니다.

> **Live:** [backlight-simulator.html](./backlight-simulator.html)  
> 별도 서버 없이 HTML 파일 하나로 동작합니다. 브라우저에서 바로 열면 됩니다.

---

## 이런 분들에게 유용해요

- 촬영 전에 "몇 시에 나가야 좋은 빛을 받을 수 있지?" 감을 잡고 싶은 분
- 역광/사이드광/순광 차이를 시각적으로 이해하고 싶은 분
- 골든아워 시간대를 계절별로 빠르게 확인하고 싶은 분
- 필 라이트(스트로브/반사판)를 언제, 얼마나 써야 하는지 연습하고 싶은 분

---

## 주요 기능

### 계절별 태양 데이터 (서울 북위 37° 기준)

| 계절 | 일출 ~ 일몰 | 정오 최고 고도 | 골든아워 |
|------|------------|--------------|---------|
| 🌸 봄 | 06:00 ~ 18:30 | 52° | 06:00~07:30 / 17:00~18:30 |
| ☀ 여름 | 05:12 ~ 19:48 | 76° | 05:00~07:00 / 18:00~20:00 |
| 🍂 가을 | 06:12 ~ 17:48 | 47° | 06:30~08:00 / 16:00~17:30 |
| ❄ 겨울 | 07:42 ~ 17:18 | 29° | 07:30~09:00 / 15:00~17:00 |

### 실시간 시각화

- **TOP VIEW** — 카메라·피사체·태양의 수평 관계 + 좌하단 고도 다이어그램
- **PREVIEW** — 인물에 빛이 어떻게 들어오는지 미리보기 (림라이트, 실루엣, 피부톤 변화)
- **LIGHT INFO** — 림라이트 강도, 색온도, 그림자, 추천 여부 요약

### 추천 시스템 (4단계 고도 기반)

| 고도 | 구분 | 판정 |
|------|------|------|
| 65°+ | 탑라이트 | 🚫 비추 — 어떤 방향이든 인물 비추 |
| 45~65° | 높은 태양 | ⚠ 주의 — 순광 비추, 역광은 필 라이트 필수 |
| 20~45° | 일반 | ✓ 좋음 — 방향별 적절한 추천 |
| 0~20° | 골든아워 | ★ 최고 — 인물 촬영 황금 시간대 |

### 컨트롤

- **시간 슬라이더 + 타임라인** — 일출~일몰 범위에서 시간 조절
- **골든아워 바로가기** — 원클릭으로 황금시간대 이동
- **태양 수평 방향** — 역광(180°) / 좌역광 / 우역광 / 사이드(90°) / 순광(0°) 프리셋
- **필 라이트** — 역광 시 스트로브/반사판 강도 시뮬레이션 (0~100%)
- **림라이트 토글** — 림라이트 시각화 ON/OFF

---

## 사용법

1. 계절 탭을 선택합니다 (봄/여름/가을/겨울)
2. 시간 슬라이더를 움직여 태양 고도를 조절합니다
3. 프리셋 버튼이나 슬라이더로 태양 방향을 설정합니다
4. 역광일 때 필 라이트 슬라이더로 보조광 효과를 확인합니다
5. 하단 SHOOTING TIP에서 현재 조건에 맞는 촬영 팁을 확인합니다
